## Usage
Simply run

```bash
# clone this repo
$ git clone git@github.com:Yhozen/simple-encryption-cli.git

# Change directory
$ cd simple-encryption-cli

# Install dependencies
$ yarn install
# or
$ npm install

# Run it 
$ yarn start
# or 
$ npm start
```

## Security

Is this truly secure? Probably not, so don't use it for anything in production.
